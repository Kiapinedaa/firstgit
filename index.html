Configuring Git

  1. Go to VS code and click Terminal. arrow then new terminal
  2. Check if git is already installed by typing "git --version" in the command line
  3. Configure git in your local computer
    • Set your username
      git config --global user.name"Name"
    • Set your email
      git config --global user.email "Email"
  4. To initialize the project directory type git init
  5. git add place a modified file on a staging area

Git add and Commit

  1. To see the status of the file in the directory type Git Status
